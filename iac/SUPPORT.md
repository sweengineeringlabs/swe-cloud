# Support Guide

## Getting Help

If you have questions or encounter issues while using the Multi-Cloud IAC framework, please use the following channels:

### 1. Documentation
Always check the [Architecture Guide](./docs/3-design/ARCHITECTURE.md) and [Installation Guide](./docs/6-deployment/installation-guide.md) first.

### 2. GitHub Issues
For bug reports and feature requests, please use [GitHub Issues](https://github.com/sweengineeringlabs/swe-cloud/issues).

### 3. Community Discussions
For broad questions about multi-cloud patterns, visit our GitHub Discussions.

## Commercial Support
For dedicated support or custom provider implementations, contact support@sweengineeringlabs.dev.
