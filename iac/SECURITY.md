# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of our Multi-Cloud infrastructure seriously. If you find a vulnerability, please do NOT open a public issue. Instead:

1. Send an email to security@sweengineeringlabs.dev
2. Include a detailed description of the vulnerability and steps to reproduce.

We will respond within 24-48 hours.
