# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a Vulnerability

We take the security of CloudKit seriously. If you find a security vulnerability, please do NOT open a public issue. Instead:

1. Email security@sweengineeringlabs.dev
2. Provide a detailed description and reproduction steps.

We will respond within 24-48 hours.
